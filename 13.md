# ✅ **Concept**

### 🔹 What is HTTP?

**HTTP** is used to transfer web pages.

- Works on top of TCP
    
- Default port → **80**
    

---

### 🔹 What this program does

- Creates a TCP connection to a web server
    
- Sends an **HTTP GET request**
    
- Receives the HTML content
    
- Saves it to a file
    

---

### 🔹 Basic HTTP Request Format

```text
GET / HTTP/1.1
Host: example.com

```

---

# 💻 **Java Program**

```java
import java.io.*;
import java.net.*;
import java.util.Scanner;

public class HTTPClient {
    public static void main(String[] args) {
        try {
            Scanner sc = new Scanner(System.in);

            System.out.print("Enter website (e.g., example.com): ");
            String host = sc.nextLine();

            // Create socket (port 80 for HTTP)
            Socket socket = new Socket(host, 80);

            // Send HTTP request
            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);
            out.println("GET / HTTP/1.1");
            out.println("Host: " + host);
            out.println("Connection: close");
            out.println();

            // Read response
            BufferedReader in = new BufferedReader(
                    new InputStreamReader(socket.getInputStream()));

            BufferedWriter fileWriter = new BufferedWriter(
                    new FileWriter("output.html"));

            String line;

            while ((line = in.readLine()) != null) {
                System.out.println(line);          // print to console
                fileWriter.write(line + "\n");    // save to file
            }

            System.out.println("\nWebpage saved as output.html");

            fileWriter.close();
            socket.close();
            sc.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```
