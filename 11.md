# ✅ **Concept**

## 🔹 CSMA/CD

**CSMA/CD**

- Used in Ethernet
    
- Steps:
    
    1. **Carrier Sense** → check if channel is free
        
    2. **Transmit** → send data
        
    3. **Collision Detection** → detect collision
        
    4. **Backoff** → wait and retransmit
        

👉 In simulation, we mimic:

- Channel busy/free
    
- Collision possibility
    

---

## 🔹 CRC (Cyclic Redundancy Check)

**Cyclic Redundancy Check**

- Used for **error detection**
    
- Sender:
    
    - Appends CRC bits to data
        
- Receiver:
    
    - Divides received data
        
    - If remainder = 0 → **No error**
        
    - Else → **Error**

# 💻 **Java Program (CSMA/CD + CRC)**

```java
import java.util.Scanner;

public class CSMACD_CRC {

    // Function to perform XOR
    static String xor(String a, String b) {
        String result = "";
        for (int i = 1; i < b.length(); i++) {
            result += (a.charAt(i) == b.charAt(i)) ? '0' : '1';
        }
        return result;
    }

    // CRC Division
    static String crcDivision(String data, String divisor) {
        int pick = divisor.length();
        String temp = data.substring(0, pick);

        while (pick < data.length()) {
            if (temp.charAt(0) == '1')
                temp = xor(divisor, temp) + data.charAt(pick);
            else
                temp = xor("0".repeat(pick), temp) + data.charAt(pick);
            pick++;
        }

        if (temp.charAt(0) == '1')
            temp = xor(divisor, temp);
        else
            temp = xor("0".repeat(pick), temp);

        return temp;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Enter data (binary): ");
        String data = sc.next();

        System.out.print("Enter divisor: ");
        String divisor = sc.next();

        // Append zeros
        String appendedData = data + "0".repeat(divisor.length() - 1);

        // Generate CRC
        String remainder = crcDivision(appendedData, divisor);
        String transmitted = data + remainder;

        System.out.println("Transmitted Data: " + transmitted);

        // CSMA/CD Simulation
        System.out.println("\n--- CSMA/CD Simulation ---");
        System.out.println("Checking channel...");

        if (Math.random() < 0.3) {
            System.out.println("Collision detected! Retrying...");
        } else {
            System.out.println("Channel free. Data sent.");
        }

        // Receiver Side
        System.out.print("\nEnter received data: ");
        String received = sc.next();

        String check = crcDivision(received, divisor);

        if (check.replace("0", "").length() == 0) {
            System.out.println("No Error Detected");
        } else {
            System.out.println("Error Detected");
        }

        sc.close();
    }
}
```

# 🧠 **Explanation (Simple)**

### Sender:

- Adds CRC bits → sends data
    

### CSMA/CD:

- Checks channel
    
- May simulate collision
    

### Receiver:

- Recomputes CRC
    
- If remainder = 0 → correct
    
- Else → error

# 🎯 **Important Points for Exam**

- CSMA/CD avoids collisions
    
- CRC detects transmission errors
    
- Works at **Data Link Layer**
    
- CRC is **more reliable than parity**
