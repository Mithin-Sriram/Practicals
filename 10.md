# ✅ **Concept (Link State Routing)**

### 🔹 What is Link State Routing?

**Link State Routing** is used by routers to compute the **shortest path** using full network knowledge.

👉 Each router:

- Knows the **entire network topology**
    
- Applies **Dijkstra’s Algorithm**
    

---

### 🔹 Key Idea

- Build a graph of the network
    
- Compute shortest path from source to all nodes
    
- Select best path to destination (server)

# 💻 **Java Program (Dijkstra’s Algorithm)**

```java
import java.util.Scanner;

public class LinkStateRouting {
    static final int MAX = 10;
    static final int INF = 999;

    public static void main(String[] args) {
        int n;
        int[][] cost = new int[MAX][MAX];
        int[] dist = new int[MAX];
        boolean[] visited = new boolean[MAX];
        int[] parent = new int[MAX];

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter number of nodes: ");
        n = sc.nextInt();

        System.out.println("Enter cost matrix:");
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                cost[i][j] = sc.nextInt();
            }
        }

        int source, destination;
        System.out.print("Enter source (client node): ");
        source = sc.nextInt();

        System.out.print("Enter destination (server node): ");
        destination = sc.nextInt();

        // Initialization
        for (int i = 0; i < n; i++) {
            dist[i] = cost[source][i];
            parent[i] = source;
            visited[i] = false;
        }

        dist[source] = 0;
        visited[source] = true;

        // Dijkstra's Algorithm
        for (int count = 1; count < n - 1; count++) {
            int min = INF, nextNode = -1;

            for (int i = 0; i < n; i++) {
                if (!visited[i] && dist[i] < min) {
                    min = dist[i];
                    nextNode = i;
                }
            }

            visited[nextNode] = true;

            for (int i = 0; i < n; i++) {
                if (!visited[i] && (min + cost[nextNode][i] < dist[i])) {
                    dist[i] = min + cost[nextNode][i];
                    parent[i] = nextNode;
                }
            }
        }

        // Print shortest path
        System.out.println("\nShortest distance: " + dist[destination]);

        System.out.print("Path: ");
        printPath(parent, source, destination);
        System.out.println();

        sc.close();
    }

    static void printPath(int[] parent, int source, int j) {
        if (j == source) {
            System.out.print(source + " ");
            return;
        }
        printPath(parent, source, parent[j]);
        System.out.print(j + " ");
    }
}
```

# 🧠 **Explanation (Simple)**

- `cost[][]` → network graph
    
- `dist[]` → shortest distances
    
- `visited[]` → processed nodes
    
- `parent[]` → stores path
    

### Working:

1. Start from source
    
2. Pick nearest node
    
3. Update neighbors
    
4. Repeat until destination reached
    

---

# 🎯 **Path Analysis (Client → Server)**

Example:

- Direct path: `0 → 2 = 999 (not valid)`
    
- Best path: `0 → 1 → 2 = 5`
    

👉 Algorithm finds **optimal route automatically**
