
# ✅ **Concept (What the question means)**

This is a **UDP Echo Client–Server** program.

### 🔹 What is UDP?

UDP (User Datagram Protocol) is:

- **Connectionless**
    
- **Faster but unreliable**
    
- Data is sent as **datagrams (packets)**
    

---

### 🔹 Echo Server Idea

- Receives a message from client
    
- Sends the **same message back** (echo)
    

### 🔹 Echo Client Idea

- Sends a message to server
    
- Receives echoed message
    
- **Verifies** if sent = received
    

---

### 🔑 Key Difference from TCP:

- No connection (`accept()` not used)
    
- Uses `DatagramSocket` and `DatagramPacket`
    

---

# 💻 **Java Program**

## 🔹 UDP Echo Server

```java
import java.net.*;

public class UDPEchoServer {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket(5000);
            byte[] buffer = new byte[1024];

            System.out.println("UDP Server is running...");

            // Receive packet
            DatagramPacket request = new DatagramPacket(buffer, buffer.length);
            socket.receive(request);

            String received = new String(request.getData(), 0, request.getLength());
            System.out.println("Received: " + received);

            // Send same data back (echo)
            DatagramPacket response = new DatagramPacket(
                    request.getData(),
                    request.getLength(),
                    request.getAddress(),
                    request.getPort()
            );

            socket.send(response);

            socket.close();
            System.out.println("Response sent, server closed");

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

---

## 🔹 UDP Echo Client

```java
import java.net.*;

public class UDPEchoClient {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket();

            String message = "hello_udp";
            byte[] sendData = message.getBytes();

            InetAddress ip = InetAddress.getByName("localhost");

            // Send packet
            DatagramPacket request = new DatagramPacket(sendData, sendData.length, ip, 5000);
            socket.send(request);

            // Receive echo
            byte[] buffer = new byte[1024];
            DatagramPacket response = new DatagramPacket(buffer, buffer.length);
            socket.receive(response);

            String received = new String(response.getData(), 0, response.getLength());

            System.out.println("Sent: " + message);
            System.out.println("Received: " + received);

            // Verification
            if (message.equals(received)) {
                System.out.println("Verification: SUCCESS (Same message)");
            } else {
                System.out.println("Verification: FAILED (Different message)");
            }

            socket.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

# 🧠 **Explanation (Simple)**

### Server:

- `DatagramSocket(5000)` → listens on port
    
- `receive()` → gets data from client
    
- Uses client IP + port → sends response
    
- Sends same message back
    

---

### Client:

- Sends `"hello_udp"`
    
- Waits for response
    
- Compares sent vs received
    
- Prints verification result
    

---

# 🎓 **Important Points for Exam**

- UDP has **no connection setup**
    
- Data may be **lost or unordered**
    
- Faster than TCP
    
- Uses **DatagramSocket & DatagramPacket**
