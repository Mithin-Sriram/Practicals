# ✅ **Concept**

### 🔹 What is Echo Client–Server?

In an **echo system**:

- Client sends a message
    
- Server sends **the same message back**
    

Uses **Transmission Control Protocol**:

- Reliable
    
- Connection-oriented
    

---

### 🔹 Working Flow

1. Server starts and waits
    
2. Client connects
    
3. Client sends message
    
4. Server receives and echoes
    
5. Client displays response
    

---

# 💻 **Java Program**

## 🔹 Server Program

```java
import java.io.*;
import java.net.*;

public class TCPEchoServer {
    public static void main(String[] args) {
        try {
            ServerSocket serverSocket = new ServerSocket(5000);
            System.out.println("Server waiting...");

            Socket socket = serverSocket.accept();
            System.out.println("Client connected");

            BufferedReader in = new BufferedReader(
                    new InputStreamReader(socket.getInputStream()));

            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

            String message;

            while ((message = in.readLine()) != null) {
                System.out.println("Received: " + message);

                // Echo back
                out.println(message);
            }

            socket.close();
            serverSocket.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

---

## 🔹 Client Program

```java
import java.io.*;
import java.net.*;
import java.util.Scanner;

public class TCPEchoClient {
    public static void main(String[] args) {
        try {
            Socket socket = new Socket("localhost", 5000);

            BufferedReader in = new BufferedReader(
                    new InputStreamReader(socket.getInputStream()));

            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

            Scanner sc = new Scanner(System.in);

            System.out.print("Enter message: ");
            String msg = sc.nextLine();

            // Send message
            out.println(msg);

            // Receive echo
            String response = in.readLine();
            System.out.println("Echo from server: " + response);

            socket.close();
            sc.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```
