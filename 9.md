# ✅ **Concept (Distance Vector Routing)**

### 🔹 What it does

**Distance Vector Routing** helps routers find the **shortest path** to all other nodes.

- Each node keeps a **routing table**
    
- Shares it with neighbors
    
- Updates using **Bellman-Ford logic**
    

---

### 🔹 Formula used

The core idea is:

[  
D_x(y) = \min { C(x,v) + D_v(y) }  
]

👉 Distance from node **x → y** via neighbor **v**

---

# 💻 **Java Program**

```java
import java.util.Scanner;

public class DistanceVectorRouting {
    static final int MAX = 10;
    static final int INF = 999;

    public static void main(String[] args) {
        int n;
        int[][] cost = new int[MAX][MAX];
        int[][] dist = new int[MAX][MAX];

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter number of nodes: ");
        n = sc.nextInt();

        System.out.println("Enter cost matrix:");
        for (int i = 0; i < n; i++) {
            for (int j = 0; j < n; j++) {
                cost[i][j] = sc.nextInt();

                // Initialize distance table
                dist[i][j] = cost[i][j];
            }
        }

        // Distance Vector Algorithm
        for (int k = 0; k < n; k++) {
            for (int i = 0; i < n; i++) {
                for (int j = 0; j < n; j++) {
                    if (dist[i][j] > cost[i][k] + dist[k][j]) {
                        dist[i][j] = cost[i][k] + dist[k][j];
                    }
                }
            }
        }

        // Display shortest paths
        System.out.println("\nShortest distance matrix:");
        for (int i = 0; i < n; i++) {
            System.out.print("From node " + i + ": ");
            for (int j = 0; j < n; j++) {
                System.out.print(dist[i][j] + " ");
            }
            System.out.println();
        }

        // Find path from client to server
        int source, destination;
        System.out.print("\nEnter source (client node): ");
        source = sc.nextInt();

        System.out.print("Enter destination (server node): ");
        destination = sc.nextInt();

        System.out.println("Shortest distance from " + source + " to " + destination + " is: " + dist[source][destination]);

        sc.close();
    }
}
```

# 🧠 **Explanation (Simple)**

- `cost[][]` → initial distances
    
- `dist[][]` → updated shortest distances
    
- Triple loop → applies Bellman-Ford logic
    
- Updates if a shorter path is found
    

👉 Example:  
Instead of going directly `0 → 2 (cost 4)`  
It finds better path `0 → 1 → 2 (cost 3)`

# 🎯 **Important Points for Exam**

- Works on **distributed routing**
    
- Uses **neighbor information**
    
- Iteratively updates routes
    
- May suffer from **count-to-infinity problem**
