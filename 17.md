# ✅ **Concept**

### 🔹 What is CRC?

**Cyclic Redundancy Check** is an error-detection technique used in data communication.

---

### 🔹 Working

1. Sender:
    
    - Appends zeros to data
        
    - Performs binary division using divisor
        
    - Gets **remainder (CRC bits)**
        
    - Sends **data + CRC**
        
2. Receiver:
    
    - Divides received data
        
    - If remainder = **0 → No error**
        
    - Else → **Error detected**
        

---

# 💻 **Java Program**

```java
import java.util.Scanner;

public class CRC {

    // XOR operation
    static String xor(String a, String b) {
        String result = "";
        for (int i = 1; i < b.length(); i++) {
            result += (a.charAt(i) == b.charAt(i)) ? '0' : '1';
        }
        return result;
    }

    // CRC division
    static String divide(String dividend, String divisor) {
        int pick = divisor.length();
        String temp = dividend.substring(0, pick);

        while (pick < dividend.length()) {
            if (temp.charAt(0) == '1')
                temp = xor(divisor, temp) + dividend.charAt(pick);
            else
                temp = xor("0".repeat(pick), temp) + dividend.charAt(pick);
            pick++;
        }

        if (temp.charAt(0) == '1')
            temp = xor(divisor, temp);
        else
            temp = xor("0".repeat(pick), temp);

        return temp;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        System.out.print("Enter data: ");
        String data = sc.next();

        System.out.print("Enter divisor: ");
        String divisor = sc.next();

        // Sender side
        String appended = data + "0".repeat(divisor.length() - 1);
        String remainder = divide(appended, divisor);

        String transmitted = data + remainder;
        System.out.println("Transmitted Data: " + transmitted);

        // Receiver side
        System.out.print("\nEnter received data: ");
        String received = sc.next();

        String check = divide(received, divisor);

        if (check.replace("0", "").length() == 0) {
            System.out.println("No Error Detected");
        } else {
            System.out.println("Error Detected in Transmission");
        }

        sc.close();
    }
}
```

# 🧠 **Explanation (Simple)**

- Binary division is done using XOR
    
- Remainder = CRC bits
    
- Receiver checks remainder
    
- Non-zero → error
    

---

# 🎯 **Important Exam Points**

- CRC is more reliable than parity
    
- Uses polynomial division
    
- Detects burst errors
    
- Used in data link layer
