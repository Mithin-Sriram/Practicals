# ✅ **Concept (ARP Protocol Simulation)**

### 🔹 What is ARP?

**Address Resolution Protocol** is used to map:

```
IP Address  →  MAC Address
```

### 🔹 Why ARP is needed?

- Devices communicate using **MAC addresses** in a local network
    
- But users/applications use **IP addresses**  
    👉 ARP converts IP → MAC
    

---

### 🔹 Real Working (Idea)

1. Sender checks ARP cache
    
2. If not found → broadcasts ARP request
    
3. Target replies with its MAC address
    
4. Entry stored in ARP table
    

---

### 🔹 What we simulate

- User enters an IP address
    
- Program checks a **predefined table (like ARP cache)**
    
- Returns corresponding MAC address
    

---

# 💻 **Java Program (ARP Simulation)**

```java
import java.util.HashMap;
import java.util.Scanner;

public class ARPSimulation {
    public static void main(String[] args) {

        // Simulated ARP Table (IP → MAC)
        HashMap<String, String> arpTable = new HashMap<>();
        arpTable.put("192.168.1.1", "AA:BB:CC:DD:EE:01");
        arpTable.put("192.168.1.2", "AA:BB:CC:DD:EE:02");
        arpTable.put("192.168.1.3", "AA:BB:CC:DD:EE:03");
        arpTable.put("192.168.1.4", "AA:BB:CC:DD:EE:04");

        Scanner sc = new Scanner(System.in);

        System.out.print("Enter IP Address: ");
        String ip = sc.nextLine();

        if (arpTable.containsKey(ip)) {
            System.out.println("MAC Address: " + arpTable.get(ip));
        } else {
            System.out.println("IP Address not found in ARP Table");
        }

        sc.close();
    }
}
```

# 🎓 **Important Points for Exam**

- ARP works only in **local network (LAN)**
    
- Uses **broadcast request + unicast reply**
    
- Maintains **ARP cache/table**
    
- Maps **IP → MAC**

# 🎯 **Viva Questions**

### 1. What is ARP?

Protocol used to map **IP address to MAC address**.

---

### 2. What is ARP cache?

A table storing recent IP–MAC mappings.

---

### 3. Is ARP request broadcast or unicast?

Broadcast.

---

### 4. Is ARP reply broadcast or unicast?

Unicast.

---

### 5. What happens if MAC is not found?

ARP request is sent in network.

---

### 6. Which layer does ARP work on?

Between **Network and Data Link layer**.
