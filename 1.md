# ✅ **Concept (What the question means)**

This is a **TCP client-server communication** program.

- A **server**:
    
    - Listens on a port
        
    - Accepts a connection from a client
        
    - Reads data sent by the client
        
    - Prints it
        
    - Closes connection
        
- A **client**:
    
    - Connects to server
        
    - Sends the message `"hello_client"`
        

### Key Idea:

TCP is **connection-oriented**, meaning:

1. Connection is established
    
2. Data is transmitted reliably
    
3. Connection is closed
    

---

# 💻 **Java Program**

## 🔹 Server Program

```java
import java.io.*;
import java.net.*;

public class TCPServer {
    public static void main(String[] args) {
        try {
            ServerSocket serverSocket = new ServerSocket(5000);
            System.out.println("Server is waiting for connection...");

            Socket socket = serverSocket.accept();
            System.out.println("Client connected");

            BufferedReader in = new BufferedReader(
                    new InputStreamReader(socket.getInputStream()));

            String message;
            while ((message = in.readLine()) != null) {
                System.out.println("Received: " + message);
            }

            socket.close();
            serverSocket.close();
            System.out.println("Connection closed");

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

## 🔹 Client Program

```java
import java.io.*;
import java.net.*;

public class TCPClient {
    public static void main(String[] args) {
        try {
            Socket socket = new Socket("localhost", 5000);

            PrintWriter out = new PrintWriter(socket.getOutputStream(), true);

            out.println("hello_client");

            socket.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

# 🧠 **Explanation (Line by Line – Simple)**

### Server:

- `ServerSocket(5000)` → creates server on port 5000
    
- `accept()` → waits for client
    
- `getInputStream()` → receives data
    
- `readLine()` → reads message
    
- loop → reads until client disconnects
    
- closes socket
    

### Client:

- `Socket("localhost", 5000)` → connects to server
    
- `getOutputStream()` → sends data
    
- `println("hello_client")` → sends message
    
- closes connection

# 🎓 **Viva Questions & Answers**

### 1. What is TCP?

TCP (Transmission Control Protocol) is a **connection-oriented protocol** that ensures reliable data transfer.

---

### 2. Difference between TCP and UDP?

|TCP|UDP|
|---|---|
|Reliable|Unreliable|
|Connection-oriented|Connectionless|
|Slower|Faster|

---

### 3. What is a socket?

A socket is an **endpoint for communication** between client and server.

---

### 4. What does `ServerSocket` do?

It listens for incoming client connections.

---

### 5. What does `accept()` do?

It **blocks (waits)** until a client connects.

---

### 6. Why use port number 5000?

Ports identify services. 5000 is just a **user-defined port**.

---

### 7. What is `localhost`?

It refers to the **same machine (127.0.0.1)**.

---

### 8. Why loop while reading?

To read **all data until client closes connection**.

---

### 9. What happens if server is not started first?

Client will throw **connection refused error**.

---

### 10. Is TCP full duplex?

Yes, both client and server can send and receive simultaneously.
