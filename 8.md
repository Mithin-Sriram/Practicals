# ✅ **Concept (Java-based Congestion Control Simulation)**

We simulate **TCP congestion control**:

**TCP Congestion Control**

### Phases:

1. **Slow Start**

   * cwnd (congestion window) doubles
2. **Congestion Avoidance**

   * cwnd increases linearly
3. **Packet Loss**

   * cwnd reduces
   * threshold updated

---

# 💻 **Java Program**

```java
import java.util.Scanner;

public class CongestionControlSimulation {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int cwnd = 1;              // congestion window
        int threshold = 16;       // slow start threshold
        int maxRounds;

        System.out.print("Enter number of transmission rounds: ");
        maxRounds = sc.nextInt();

        System.out.println("\nRound\tcwnd\tThreshold\tStatus");

        for (int i = 1; i <= maxRounds; i++) {

            // Simulate packet loss randomly
            boolean loss = Math.random() < 0.3;

            if (loss) {
                System.out.println(i + "\t" + cwnd + "\t" + threshold + "\tPacket Loss");

                threshold = cwnd / 2;
                cwnd = 1;  // reset to slow start
            } else {
                if (cwnd < threshold) {
                    // Slow Start
                    System.out.println(i + "\t" + cwnd + "\t" + threshold + "\tSlow Start");
                    cwnd = cwnd * 2;
                } else {
                    // Congestion Avoidance
                    System.out.println(i + "\t" + cwnd + "\t" + threshold + "\tCongestion Avoidance");
                    cwnd = cwnd + 1;
                }
            }
        }

        sc.close();
    }
}
```
