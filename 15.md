# ✅ **Concept**

### 🔹 What is UDP Echo?

In an echo system:

- Client sends a message
    
- Server returns **the same message**
    

Uses **User Datagram Protocol**:

- Connectionless
    
- Faster (but no guarantee of delivery)
    

---

### 🔹 Working Flow

1. Server waits for datagram
    
2. Client sends message
    
3. Server receives it
    
4. Server sends same message back
    
5. Client displays echoed message
    

---

# 💻 **Java Program**

## 🔹 UDP Echo Server

```java
import java.net.*;

public class UDPEchoServer {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket(5000);
            byte[] buffer = new byte[1024];

            System.out.println("UDP Server running...");

            while (true) {
                // Receive message
                DatagramPacket request = new DatagramPacket(buffer, buffer.length);
                socket.receive(request);

                String msg = new String(request.getData(), 0, request.getLength());
                System.out.println("Received: " + msg);

                // Echo back
                DatagramPacket response = new DatagramPacket(
                        request.getData(),
                        request.getLength(),
                        request.getAddress(),
                        request.getPort()
                );

                socket.send(response);
            }

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

## 🔹 UDP Echo Client

```java
import java.net.*;
import java.util.Scanner;

public class UDPEchoClient {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket();
            Scanner sc = new Scanner(System.in);

            System.out.print("Enter message: ");
            String message = sc.nextLine();

            byte[] sendData = message.getBytes();
            InetAddress ip = InetAddress.getByName("localhost");

            // Send message
            DatagramPacket request = new DatagramPacket(sendData, sendData.length, ip, 5000);
            socket.send(request);

            // Receive echo
            byte[] buffer = new byte[1024];
            DatagramPacket response = new DatagramPacket(buffer, buffer.length);
            socket.receive(response);

            String received = new String(response.getData(), 0, response.getLength());

            System.out.println("Echo from server: " + received);

            socket.close();
            sc.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

# 🎓 **Viva Questions**

### 1. What is UDP?

Connectionless protocol for fast communication.

---

### 2. Difference between TCP and UDP?

|TCP|UDP|
|---|---|
|Reliable|Unreliable|
|Connection-oriented|Connectionless|

---

### 3. What is DatagramPacket?

Packet used to send/receive data in UDP.

---

### 4. Why UDP is faster?

No connection setup or acknowledgment.

---

### 5. Can packets be lost?

Yes, UDP does not guarantee delivery.
