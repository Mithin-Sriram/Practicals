This question is a bit oddly worded (it mixes **DNS** with **frames**), but in exams it usually means:

👉 Build a **UDP-based client–server program** where

- Client sends a **domain name + frame size**
    
- Server acts like a **simple DNS** (maps domain → IP)
    
- Server **splits the response into frames** (chunks) based on the given size
    
- Sends frames back using UDP
    

---

# ✅ **Concept (DNS using UDP + Framing)**

### 🔹 DNS Idea:

DNS (Domain Name System) maps:

```
www.google.com → 142.250.x.x
```

Real DNS uses **UDP (port 53)** because:

- Faster
    
- No connection overhead
    

---

### 🔹 What we simulate:

1. Client sends:
    
    - Domain name
        
    - Frame size
        
2. Server:
    
    - Looks up IP (using Java `InetAddress`)
        
    - Converts IP to string
        
    - Splits into **frames of given size**
        
    - Sends each frame separately
        
3. Client:
    
    - Receives all frames
        
    - Combines them
        
    - Displays final result
        

---

# 💻 **Java Program**

## 🔹 Server Program

```java
import java.net.*;

public class DNSServer {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket(5000);
            byte[] buffer = new byte[1024];

            System.out.println("DNS Server running...");

            // Receive request
            DatagramPacket request = new DatagramPacket(buffer, buffer.length);
            socket.receive(request);

            String input = new String(request.getData(), 0, request.getLength());
            String[] parts = input.split(",");

            String domain = parts[0];
            int frameSize = Integer.parseInt(parts[1]);

            System.out.println("Requested Domain: " + domain);
            System.out.println("Frame Size: " + frameSize);

            // Get IP
            InetAddress ip = InetAddress.getByName(domain);
            String ipStr = ip.getHostAddress();

            System.out.println("Resolved IP: " + ipStr);

            // Send in frames
            for (int i = 0; i < ipStr.length(); i += frameSize) {
                String frame = ipStr.substring(i, Math.min(i + frameSize, ipStr.length()));

                byte[] sendData = frame.getBytes();
                DatagramPacket response = new DatagramPacket(
                        sendData,
                        sendData.length,
                        request.getAddress(),
                        request.getPort()
                );

                socket.send(response);
            }

            // Send END signal
            byte[] end = "END".getBytes();
            DatagramPacket endPacket = new DatagramPacket(
                    end, end.length, request.getAddress(), request.getPort()
            );
            socket.send(endPacket);

            socket.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

## 🔹 Client Program

```java
import java.net.*;
import java.util.Scanner;

public class DNSClient {
    public static void main(String[] args) {
        try {
            DatagramSocket socket = new DatagramSocket();
            Scanner sc = new Scanner(System.in);

            System.out.print("Enter domain (e.g., google.com): ");
            String domain = sc.nextLine();

            System.out.print("Enter frame size: ");
            int frameSize = sc.nextInt();

            String message = domain + "," + frameSize;
            byte[] sendData = message.getBytes();

            InetAddress ip = InetAddress.getByName("localhost");

            // Send request
            DatagramPacket request = new DatagramPacket(sendData, sendData.length, ip, 5000);
            socket.send(request);

            // Receive frames
            byte[] buffer = new byte[1024];
            StringBuilder result = new StringBuilder();

            while (true) {
                DatagramPacket response = new DatagramPacket(buffer, buffer.length);
                socket.receive(response);

                String frame = new String(response.getData(), 0, response.getLength());

                if (frame.equals("END"))
                    break;

                result.append(frame);
            }

            System.out.println("Resolved IP Address: " + result.toString());

            socket.close();

        } catch (Exception e) {
            System.out.println(e);
        }
    }
}
```

# 🧠 **Explanation (Simple)**

- Client sends: `"google.com,4"`
    
- Server resolves IP
    
- Suppose IP = `"142.250.183.14"`
    
- Frames (size 4):
    
    ```
    142.
    250.
    183.
    14
    ```
    
- Client receives and combines → final IP
    

---

# 🎓 **Important Points for Exam**

- DNS uses **UDP (fast, connectionless)**
    
- Frames simulate **packet segmentation**
    
- `InetAddress.getByName()` resolves domain
    
- End signal is needed to stop receiving loop

# 🎯 **Viva Questions**

### 1. Why does DNS use UDP?

Because it is **faster and lightweight**.

---

### 2. What is framing?

Breaking data into **smaller chunks for transmission**.

---

### 3. What is `InetAddress.getByName()`?

It resolves a **domain name to IP address**.

---

### 4. Why send "END"?

To indicate **no more frames**.

---

### 5. What happens if frame size is large?

Less number of packets are sent.

---

### 6. Is UDP reliable?

No, packets can be **lost or unordered**.
